# CRM Management System using MySQL

##  Overview

The CRM (Customer Relationship Management) System is a MySQL database project designed to manage customers, products, employees, sales, interactions, support tickets, tasks, feedback, and leads. The project demonstrates relational database design and SQL-based business reporting.

##  Technologies Used

- MySQL
- MySQL Workbench
- SQL

##  Project Files

- **01_CRM_Tables.sql** – Database and table creation
- **02_CRM_Insert_Data.sql** – Sample data insertion
- **03_CRM_Business_Queries.sql** – Business reports and SQL queries

##  SQL Concepts Used

- DDL (CREATE DATABASE, CREATE TABLE)
- DML (INSERT, UPDATE, DELETE)
- SELECT Queries
- WHERE, ORDER BY
- Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)
- GROUP BY & HAVING
- INNER JOIN
- Subqueries

##  Business Reports

The project includes SQL queries for:

- Customer Purchase History
- Product Sales Report
- Employee Performance Report
- Revenue Analysis
- Customer Interaction Report
- Support Ticket Report
- Customer Feedback Analysis
- Lead Source & Status Analysis
- CRM Dashboard Summary

## How to Run

1. Execute **01_CRM_Tables.sql**
2. Execute **02_CRM_Insert_Data.sql**
3. Execute **03_CRM_Business_Queries.sql**

## Learning Outcomes

- Designed a relational database with multiple related tables.
- Implemented primary and foreign key relationships.
- Performed CRUD operations.
- Generated business reports using SQL joins, aggregate functions, grouping, and subqueries.
